# 🐧 企鹅大乱斗 (Penguin Battle)

一个基于 Vue 3 的回合制企鹅格斗游戏，集成在个人作品集网站中。

## 🎮 游戏特性

### 核心战斗系统
- **27 个独特技能** — 4 个稀有度层级（T0神技/T1核心/T2输出/T3辅助），6 大类别（属性/增幅/攻击/控制/防御/特殊）
- **25 把武器** — 4 种类型（小型/中型/大型/暗器），5 种品质（凡器→真·神器），支持进化和强化
- **PRD 伪随机系统** — 暴击/闪避使用伪随机分布，连续未触发时概率递增，消除"脸黑"体验
- **能量终结技** — 普攻+10能量，受击+5能量，满100自动释放终结技
- **权重随机 AI** — 技能/武器选择使用加权轮盘赌算法，低血量优先治疗，高伤害技能权重更高

### 成长系统
- **等级系统** — 最高等级 50，经验公式 `floor(100 × 1.15^(level-1))`
- **技能获取** — 每 2 级随机获得 1 个技能或武器
- **4 大职业** — 剑魂(力量)/影刺(敏捷)/金刚(生命)/风行者(速度)，Lv15 解锁，Lv40 觉醒
- **8 大境界** — 练气→筑基→金丹→元婴→化神→渡劫→大乘→飞升
- **称号系统** — 10 个等级称号，从"初心者"到"一代宗师"
- **10 个成就** — 战斗胜利/连胜/等级/收集/挑战塔等

### 玩法模式
- **随机乐斗** — 匹配与你实力相当的随机企鹅，使用和玩家一样的技能/武器池
- **连胜奖励** — 连胜 3/5/7/10 场触发精英/传说/神话/远古级特殊对手
- **斗神塔** — 100 层挑战，每日 6 次，10 层后可扫荡
- **好友系统** — 8 个 NPC 好友，亲密度系统
- **木桩训练** — 可配置血量/闪避/格挡，实时 DPS 统计
- **GM 后台** — 修改属性/发放物品/快捷操作

### 战斗机制
| 机制 | 说明 |
|------|------|
| 暴击 | `min(0.05 + 敏捷×0.002, 40%)`，PRD 伪随机 |
| 闪避 | `min(0.03 + 敏捷×0.001 + 速度×0.001, 35%)`，PRD 伪随机 |
| 格挡 | 固定 15%，减伤 50% |
| 反击 | `min(0.1 + 敏捷×0.001, 30%)`，反弹 50% 伤害 |
| 连击 | 武器特效，概率追加一次攻击 |
| 眩晕 | 控制技，敌人无法行动 N 回合 |
| 忽略 | 敌人被忽略，无法攻击，无视格挡 |
| 装死 | 生命归零时免死一次，恢复 1 点生命 |
| 嗜血 | 35% 概率吸取造成伤害的 50% |
| 大海无量 | 30% 概率反弹 100% 受到的伤害 |
| 能量系统 | 普攻+10，受击+5，满100释放终结技 |
| 硬终止 | 50 回合后按生命百分比→总伤害裁决 |

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 运行测试
npx vitest run

# 构建生产版本
npm run build
```

## 🧪 测试

```bash
# 运行所有测试
npx vitest run

# 运行特定测试文件
npx vitest run src/game/__tests__/engine.test.js

# 运行 Monte Carlo 模拟（1000 场战斗统计）
npx vitest run src/game/__tests__/simulation.test.js
```

**测试覆盖**：86 个测试用例，覆盖：
- 伤害公式计算
- 暴击/闪避/格挡/反击
- 所有 27 个技能效果
- 武器特效和进化
- 能量系统
- PRD 伪随机分布
- 状态效果持续时间
- 50 回合硬终止
- 端到端战斗模拟
- 1000 场 Monte Carlo 平衡性测试
