# Evolve 游戏集成计划

## 目标
将 [Evolve](https://github.com/pmotschmann/Evolve)（增量文明进化游戏）通过 iframe 嵌入到个人网站中。

## 架构决策
- **集成方式**: iframe 嵌入
- **部署方式**: Fork Evolve 到 `mafter822/Evolve`，启用 GitHub Pages
- **iframe URL**: `https://mafter822.github.io/Evolve/`

---

## 实施步骤

### Step 1: Fork Evolve 仓库
```bash
gh repo fork pmotschmann/Evolve --clone=false
```

### Step 2: 启用 GitHub Pages
在 Fork 的仓库中启用 GitHub Pages（从 main 分支部署）：
```bash
gh api repos/mafter822/Evolve/pages -X POST -f build_type=legacy -f source.branch=master -f source.path=/
```
或通过 GitHub Web UI 手动操作。

### Step 3: 创建 EvolveView.vue
新建 `src/views/EvolveView.vue`，包含全屏 iframe 嵌入 Evolve。

### Step 4: 添加路由
在 `src/router/index.js` 中添加 `/evolve` 路由。

### Step 5: 首页添加入口
在 `HomeView.vue` 的 Game Section 中，与企鹅大乱斗并列添加 Evolve 入口卡片。

---

## 涉及文件

| 文件 | 操作 |
|------|------|
| `src/views/EvolveView.vue` | **新建** — iframe 容器页面 |
| `src/router/index.js` | **修改** — 添加 `/evolve` 路由 |
| `src/views/HomeView.vue` | **修改** — Game Section 添加 Evolve 卡片 |

---

## 验证方式
1. `npm run build` 确认构建通过
2. `npm run dev` 启动开发服务器
3. 访问 `/#/evolve` 确认 Evolve 游戏通过 iframe 正常加载
4. 访问首页确认 Evolve 入口卡片显示正常
5. 测试 iframe 全屏显示、无滚动条问题
