# 企鹅大乱斗 - 当前设计方案

## 一、游戏概述

**名称**: 企鹅大乱斗  
**类型**: 回合制宠物对战游戏（PVE为主）  
**技术栈**: Vue 3 + Vite + Tailwind CSS 4  
**存储**: localStorage 自动存档  
**入口**: `/#/game`

| 增幅 (buff) | 被动增加伤害 | 2种 |

| 控制 (control) | 控制敌方 | 4种 |
| 防御 (defense) | 被动防御 | 5种 |
| 特殊 (special) | 特殊效果 | 10种 |

### 2. 技能稀有度

| 等级 | 说明 | 颜色 |
|------|------|------|
| T0 | 神技（必拿） | 红色 |
| T1 | 核心属性/被动 | 橙色 |
| T2 | 输出/控制 | 黄色 |
| T3 | 恢复/特殊 | 绿色 |

### 3. 技能获取机制

**累积池 + 权重概率**:
- 每个等级段有不同的技能池
- 按权重随机抽取新技能
- 未获得的技能会累积权重

**等级段开放节奏**:
| 等级段 | 新增技能 |
|--------|----------|
| Lv.1-4 | 5种属性技能 |
| Lv.5-9 | +5种被动/攻击 |
| Lv.10-14 | +8种防御/控制 |
| Lv.15-19 | +6种高阶技能 |
| Lv.20-24 | +3种神技 |
| Lv.30-34 | +4种终极技能 |

### 4. 技能升级

- 消耗精魄升级
- 每级消耗 = 基础消耗 × 当前等级
- 最高 Lv.10（属性）/ Lv.5（其他）

## 五、战斗系统

### 1. 战斗流程

```
开始战斗
    ↓
自动选择技能（AI决策）
    ↓
玩家攻击 → 伤害计算 → 特效处理
    ↓
敌方攻击 → 伤害计算 → 特效处理
    ↓
检查战斗结束条件
    ↓
循环或结算
```

### 2. AI技能选择逻辑

```
if 血量 < 30%:
    优先使用回复技能
elif 敌方血量 < 20%:
    优先使用控制技能
elif 60% 概率:
    使用高伤害技能
elif 30% 概率:
    使用控制技能
else:
    随机选择
```

### 3. 伤害公式

```
基础伤害 = 力量 × 武器基础伤害 / 50
技能伤害 = 基础伤害 × 技能倍率 × (1 + 武器强化加成)
最终伤害 = 技能伤害 × 随机系数(0.9~1.1) × (1 - 防御减免)

暴击率 = min(0.05 + 敏捷 × 0.002, 0.4)
暴击伤害 = 1.5倍

闪避率 = min(0.03 + 敏捷 × 0.001 + 速度 × 0.001, 0.35)

格挡率 = 15%
格挡减伤 = 50%

反击率 = min(0.1 + 敏捷 × 0.001, 0.3)
反击伤害 = 50%
```

### 4. 战斗日志格式

```
【第1回合】
野生企鹅(Lv.48)投掷【板砖】！超级无敌大便超人损失 37 点生命！
【连击】野生企鹅(Lv.48)追加一击！再损失 25 点！
超级无敌大便超人发动【缴械】！但野生企鹅(Lv.48)已无武器可夺！
（血量：超级无敌大便超人 256 | 野生企鹅(Lv.48) 100）
```

|------|----------|------|
| 1x | 1.0x | 正常速度 |
| 2x | 0.5x | 默认速度 |
| 3x | 0.33x | 快速 |
